## 信息收集：

### 子域名收集：

dnsrecon  被动 + 主动 + 区域传送 + 字典爆破  

```
dnsrecon -d example.com -t brt -D subdomains.txt
```

sublist3r 被动神器：爬搜索引擎、DNS、证书等不用字典，速度快

```
sublist3r -d example.com
```

amass  被动 + 主动 + 爬虫 + API  

```
amass enum -d example.com
```

subfinder 被动收集之王，速度快、结果多  

```
subfinder -d example.com
```

gobuster 纯暴力破解，速度极快 适合内网、可控环境

```
gobuster dns -d example.com -w subdomains.txt
```

实战推荐组合

```
subfinder -d example.com -o sub1.txt
sublist3r -d example.com -o sub2.txt
amass enum -passive -d example.com -o sub3.txt
cat sub*.txt | sort -u > alldomains.txt
```
